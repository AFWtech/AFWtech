# AFWtech - Home Appliances Sale & Service

AFWtech is a front-end website focused on **Home Appliances** sales and services.  
This website is designed similar to Justdial but is exclusively for **ACs, Refrigerators, Washing Machines, and Geysers**.

## Features

- **Home Page**
  - Search bar for services or products
  - Category list (AC, Refrigerator, Washing Machine, Geyser)

- **Service Page**
  - List of services for each appliance
  - "Call Now" button for direct contact

- **Sale Page**
  - Product listings with price
  - "Buy Now" button

- **Responsive Design**
  - Works on mobile, tablet, and desktop
- Direct call functionality via "Call Now" buttons

## Tech Stack

- HTML  
- CSS  
- JavaScript  

## Installation / Usage

1. Clone this repository or upload it to GitHub Pages / Netlify.  
2. Keep all files in the same folder (`index.html`, `service.html`, `sale.html`, `style.css`, `script.js`).  
3. Open `index.html` in your browser or use the GitHub Pages / Netlify URL.  
4. Use the "Call Now" button to contact directly via phone.
